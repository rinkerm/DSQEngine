# DSQEngine

A Python rules engine for determining driver disqualification outcomes from conviction histories using XML/XSD validation and explainable regulatory logic.

This engine was created with the following resources:

- [AAMVA ACD Manual](https://www.aamva.org/getmedia/b1fd2b7f-8040-4764-9b4e-4ca464c43a2c/AAMVA-Code-Dictionary-Manual-5-2.pdf)
- [FMCSA eCFR 383.51](https://www.ecfr.gov/current/title-49/subtitle-B/chapter-III/subchapter-B/part-383/subpart-D/section-383.51)
